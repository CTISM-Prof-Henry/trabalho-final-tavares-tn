Estrutura básica

Código PHP fica dentro de tags especiais.

<?php

?>
Variáveis

Variáveis em PHP começam com o símbolo $.

$idade = 25;
$nome = "Carlos";
$ativo = true;
Condicionais

Estruturas de decisão com if e else.

if ($idade >= 18) {
    echo "Maior de idade";
} else {
    echo "Menor de idade";
}
Conexão com banco de dados

PHP pode se conectar a bancos de dados usando MySQLi ou PDO.

$conexao = mysqli_connect("localhost", "usuario", "senha", "banco");

if (!$conexao) {
    die("Falha na conexão");
}

echo "Conectado com sucesso!";
