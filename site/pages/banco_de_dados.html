Banco de Dados

O que é um banco de dados

Um banco de dados organiza e armazena informações de forma estruturada, permitindo consultas e atualizações com segurança.

Tabelas e registros

Os dados ficam organizados em tabelas, formadas por linhas (registros) e colunas (campos).

Tabela: alunos
| id | nome   | idade |
|----|--------|-------|
| 1  | Maria  | 20    |
| 2  | João   | 22    |
Comandos SQL básicos

SQL é a linguagem usada para manipular dados.

SELECT * FROM alunos;

INSERT INTO alunos (nome, idade)
VALUES ('Ana', 19);

UPDATE alunos SET idade = 21 WHERE id = 1;

DELETE FROM alunos WHERE id = 2;
Chaves primárias e estrangeiras

A chave primária identifica cada registro. A chave estrangeira conecta tabelas relacionadas.

Tabela: cursos (id, nome)
Tabela: matriculas (id, aluno_id, curso_id)

aluno_id referencia alunos(id)
curso_id referencia cursos(id)
