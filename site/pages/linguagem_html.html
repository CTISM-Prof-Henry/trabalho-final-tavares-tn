Estrutura básica

Toda página HTML possui uma estrutura padrão com tags essenciais.

<html lang="pt-br">

        <meta charset="UTF-8">
        <title>Minha Página</title>
    </head>
    <body>
        <h1>Olá, mundo!</h1>
    </body>
</html>
