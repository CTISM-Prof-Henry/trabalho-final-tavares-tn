JavaScript

Variáveis

Declaração de variáveis com let, const e var.

let idade = 25;
const nome = "João";
var ativo = true;
Condicionais

Estruturas de decisão com if e else.

if (idade >= 18) {
    console.log("Maior de idade");
} else {
    console.log("Menor de idade");
}
Laços de Repetição

Repetição com for e while.

for (let i = 0; i < 5; i++) {
    console.log(i);
}
Funções

Funções podem ser declaradas de formas diferentes.

function soma(a, b) {
    return a + b;
}

const multiplica = (a, b) => a * b;
