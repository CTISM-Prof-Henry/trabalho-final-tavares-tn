Classes e Objetos

Classes definem modelos; objetos são instâncias dessas classes.

class Carro {
    string modelo = "Sedan";
}

Carro meuCarro = new Carro();
Encapsulamento

Protege os dados, permitindo acesso controlado por métodos.

class Conta {
    private double saldo;

    public void depositar(double valor) {
        saldo += valor;
    }
}
Herança

Uma classe pode herdar atributos e métodos de outra.

class Animal {
    public void comer() { }
}

class Cachorro : Animal {
    public void latir() { }
}
Polimorfismo

Objetos de classes diferentes podem responder ao mesmo método de formas distintas.

Animal animal1 = new Cachorro();
Animal animal2 = new Gato();

animal1.emitirSom(); // "Au au!"
animal2.emitirSom(); // "Miau!"
