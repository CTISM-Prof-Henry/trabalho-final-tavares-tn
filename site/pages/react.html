JSX

JSX permite escrever HTML dentro do JavaScript.

const elemento = (
    <div>
        <h2>Título</h2>
        <p>Parágrafo de exemplo</p>
    </div>
);
Props

Props passam dados de um componente pai para o filho.

function BoasVindas(props) {
    return <h1>Olá, {props.nome}!</h1>;
}

<BoasVindas nome="Ana" />
useState

Hook para controlar estado em componentes funcionais.

import { useState } from 'react';

function Contador() {
    const [valor, setValor] = useState(0);

    return (
        <button onClick={() => setValor(valor + 1)}>
            Clicou {valor} vezes
        </button>
    );
}
